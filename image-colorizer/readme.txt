Welcome to my Python Image Colorizer!

This program uses a pre-trained model to colorize black and white images, then allows for some basic editing of that
image. This project is to be implemented as a web application and will utilize Flask as the web framework.

SOURCES USED:
-- Richard Zhang, Phillip Isola, and Alexei A. Efros. "Colorful Image Colorization." ECCV 2016.
-- PyImageSearch
-- https://www.pyimagesearch.com/2019/02/25/black-and-white-image-colorization-with-opencv-and-deep-learning/
-- https://www.pyimagesearch.com/2019/02/04/keras-multiple-inputs-and-mixed-data/
-- DeOldify
-- PySimpleGUI's model: https://github.com/PySimpleGUI/PySimpleGUI-Photo-Colorizer
